# Logos del Carrusel de Clientes Referidos

Guarda aquí todos los logotipos de las marcas clientes que se mostrarán en el carrusel de la página de inicio.
Formatos recomendados: PNG (con fondo transparente), SVG o JPG.

Para lograr una apariencia premium:
- Se aplicará un filtro de escala de grises y opacidad en CSS para homogeneizar los diferentes logos corporativos.
- Al pasar el cursor por encima, recuperarán su color original de forma fluida.
